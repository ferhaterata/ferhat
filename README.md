## fzf 
A command-line fuzzy finder   
https://github.com/junegunn/fzf   
https://wiki.archlinux.org/index.php/Fzf  
`git clone --depth 1 https://github.com/junegunn/fzf.git ~/.fzf`
`~/.fzf/install`  
update: `cd ~/.fzf && git pull && ./install`
